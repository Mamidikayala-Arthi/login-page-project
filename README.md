# Login Page Project

This is a simple and responsive Login Page created using **HTML** and **CSS**.  
It contains a modern UI with gradient background, login form, and social login icons.

## 🚀 Features

- Clean and modern login UI
- Gradient background design
- Username and password input fields
- Forgot password link
- Login button with gradient style
- Social media login icons
- Sign up link

## 🛠 Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

login-page-project
│
├── index.html
└── login.css

## 💻 How to Run

1. Download or clone the repository
2. Open the folder
3. Double click **index.html**
4. The login page will open in your browser
